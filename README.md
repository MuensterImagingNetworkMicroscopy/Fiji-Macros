# Fiji-Macros

A collection of Fiji/ImageJ macros developed at the **Münster Imaging Network (MIN)**.

The macros in this repository were created for specific imaging workflows and analysis tasks. They are, however, freely available for anyone to use, adapt, and build upon.

---

## Available Macros

| Macro | Description |
|---|---|
| [Object-Based Colocalization Analyzer](./Object-Based-Colocalization/) | 2D object-based colocalization analysis for two-channel fluorescence images. Single image and batch processing. |

*More macros will be added over time.*

---

## Usage

Each macro has its own subfolder containing:
- The `.ijm` macro file
- A dedicated `README.md` with a step-by-step guide
- Example screenshots in `docs/screenshots/`

Refer to the individual README for installation and usage instructions.
